## Idea 1: "Neon Arcade" — Retro-Futuristic Pixel Culture

<text>

### Design Movement
Retro-futurism meets arcade culture. Inspired by late-80s/90s arcade cabinets, CRT monitors, and the neon-soaked aesthetic of vaporwave — but sharpened for modern web. Think pixel-art borders, scanline overlays, and glowing neon accents against deep dark backgrounds.

### Core Principles
1. **High contrast neon-on-dark** — Every element pops like a lit arcade screen
2. **Pixel-perfect modularity** — Components feel like game UI panels
3. **Playful irreverence** — The site doesn't take itself too seriously; it's fun
4. **Layered depth** — Multiple z-layers with glow effects create dimensional depth

### Color Philosophy
- **Background**: Deep midnight navy (#0a0e1a) — feels like staring into a CRT
- **Primary neon**: Electric cyan (#00f0ff) — the "power on" indicator
- **Secondary neon**: Hot magenta (#ff2d7b) — energy and excitement
- **Accent**: Lime green (#39ff14) — call-to-action highlights
- **Text**: Soft white (#e8e6f0) with slight blue tint for readability
- **Emotional intent**: Excitement, creativity, gaming culture belonging

### Layout Paradigm
Asymmetric panel-based layout. Content is organized in floating "panels" that resemble game UI windows, with slight rotation and offset. Navigation is a persistent top bar styled like an arcade menu. Portfolio grids use masonry with hover-zoom effects.

### Signature Elements
1. **Scanline overlay** — Subtle horizontal lines across the entire viewport, like a CRT screen
2. **Neon glow borders** — Key cards and buttons have animated glow pulses
3. **Pixel-art decorative corners** — Small pixel sprites as section dividers

### Interaction Philosophy
Hover states trigger neon "power-up" glows. Clicks produce subtle flash effects. Navigation transitions feel like switching channels on a retro TV. Everything feels responsive and alive.

### Animation
- Page transitions: horizontal slide with brief static/glitch effect
- Hover: elements scale up slightly with intensifying glow
- Scroll: sections fade in with upward drift
- Loading states: pixel-art progress bars

### Typography System
- **Display**: "Press Start 2P" (Google Fonts) — for headings and nav, pure arcade energy
- **Body**: "Space Grotesk" — geometric sans-serif, modern but techy
- **Hierarchy**: Display at 2rem+ for titles, Space Grotesk at 1rem for body, monospace for metadata

## Idea 2: "Ink & Paper" — Illustrative Editorial

<text>

### Design Movement
Editorial illustration meets zine culture. Inspired by independent art magazines, hand-drawn aesthetics, and the tactile quality of printed matter. The site feels like flipping through a beautifully curated artist's lookbook — with torn paper edges, ink splatter textures, and a warm, inviting palette.

### Core Principles
1. **Tactile warmth** — Every surface feels like it has texture: paper grain, ink bleed, soft edges
2. **Curated editorial flow** — Content is paced like a magazine spread, not a dashboard
3. **Artist-first hierarchy** — The art is always the loudest element; UI recedes gracefully
4. **Organic asymmetry** — Nothing is perfectly aligned; slight offsets feel hand-placed

### Color Philosophy
- **Background**: Warm cream (#faf5eb) — like aged cotton paper
- **Primary**: Rich charcoal ink (#2a2a2e) — bold, authoritative
- **Accent warm**: Burnt sienna (#c45a3c) — like a red ink stamp
- **Accent cool**: Dusty teal (#4a8b8d) — subtle contrast for links/highlights
- **Muted**: Warm gray (#9e9589) — for secondary text and dividers
- **Emotional intent**: Warmth, craftsmanship, artistic credibility, approachability

### Layout Paradigm
Magazine-spread layout with alternating full-bleed images and text columns. Sections use overlapping layers — images slightly overlap text blocks, creating depth. The portfolio uses a staggered two-column layout where items are offset vertically, like a curated gallery wall.

### Signature Elements
1. **Torn paper edge dividers** — Section breaks use SVG torn-paper shapes instead of straight lines
2. **Ink splatter accents** — Small decorative ink drops near headings and CTAs
3. **Hand-drawn underlines** — Key text uses wavy, hand-drawn-style underlines via SVG paths

### Interaction Philosophy
Interactions feel gentle and organic. Hover states reveal content like peeling back a page corner. Transitions are smooth fades, never jarring. The experience rewards slow, deliberate browsing — like savoring an art book.

### Animation
- Page transitions: soft crossfade with slight vertical shift
- Hover: images lift with soft shadow expansion, like picking up a print
- Scroll: parallax on background textures, content slides in from alternating sides
- Loading: ink drop expanding animation

### Typography System
- **Display**: "Playfair Display" — elegant serif with high contrast strokes, editorial authority
- **Body**: "Source Sans 3" — clean humanist sans-serif, excellent readability
- **Accent**: "Caveat" — handwritten style for annotations, quotes, and personal touches
- **Hierarchy**: Playfair at 3rem+ for hero titles, Source Sans at 1rem body, Caveat for captions

## Idea 3: "Digital Canvas" — Bold Geometric Expressionism

<text>

### Design Movement
Neo-brutalist expressionism with a digital art twist. Inspired by bold graphic design posters, Bauhaus geometry, and the raw energy of street art. The site uses oversized typography, stark color blocks, and dramatic negative space to create a gallery-like experience that screams creative confidence.

### Core Principles
1. **Unapologetic boldness** — Oversized elements, stark contrasts, nothing subtle
2. **Geometric structure** — Hard angles, clean blocks, mathematical precision in layout
3. **Negative space as statement** — Empty space is intentional and powerful
4. **Raw authenticity** — No glossy gradients; flat colors, hard shadows, honest materials

### Color Philosophy
- **Background**: Off-black (#111111) with warm undertone — gallery wall darkness
- **Primary**: Vivid coral (#FF6B4A) — warm, energetic, impossible to ignore
- **Secondary**: Electric yellow (#FFD23F) — highlights and accents that demand attention
- **Tertiary**: Cool slate (#3D5A80) — for depth and contrast panels
- **Text**: Near-white (#F0EDE8) — warm white that doesn't strain against dark backgrounds
- **Emotional intent**: Confidence, creative authority, modern edge, professional boldness

### Layout Paradigm
Oversized grid with dramatic scale shifts. Hero sections use full-viewport typography. Content alternates between massive single-focus blocks and tight multi-column grids. The portfolio uses a bold bento-grid layout where featured pieces get 2x or 3x the space of others. Sections are separated by thick color-block dividers rather than whitespace.

### Signature Elements
1. **Oversized floating typography** — Key words break out of their containers, overlapping images
2. **Hard-edge color blocks** — Sections use stark, full-width color transitions (no gradients)
3. **Geometric accent shapes** — Circles, triangles, and rectangles used as decorative overlays

### Interaction Philosophy
Interactions are snappy and decisive. Hover states use instant color inversions or bold underline reveals. Cursor interactions feel weighty — elements respond with confidence, not timidity. Navigation clicks trigger bold slide transitions.

### Animation
- Page transitions: hard horizontal wipe with color block overlay
- Hover: instant color inversion on buttons, bold underline slide on links
- Scroll: elements enter with strong upward thrust, no gentle fading
- Cursor: custom cursor that changes shape based on interactive context
- Stagger: grid items enter one-by-one with rhythmic timing

### Typography System
- **Display**: "Syne" — bold, geometric, with distinctive character shapes that feel artistic
- **Body**: "DM Sans" — clean geometric sans-serif, modern and highly readable
- **Hierarchy**: Syne at 4rem+ for hero statements (can go up to 8rem), DM Sans at 1rem body
- **Special**: Syne Extra Bold for single-word impact statements that overlap sections
