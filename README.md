# Artist Portfolio

Cleaned Vite + React + TypeScript portfolio site.

## Structure

- `src/components/layout` — navbar, footer, layout shell
- `src/components/shared` — reusable content components
- `src/pages/home` — home page
- `src/pages/portfolio` — portfolio page
- `src/pages/services` — service pages like emotes and branding
- `src/pages/about` — about page
- `src/pages/contact` — contact page
- `src/pages/store` — store page
- `src/pages/system` — not found and system pages
- `src/styles` — global theme tokens and global styles
- `server` — optional static file host for production
- `docs` — notes and planning docs

## Notes

This version removes most generator bloat, trims unused UI files, centralizes theme tokens, and uses a more standard page/layout structure.


## Theme

This project now uses a Fire / Flare / Flareon-inspired global theme defined in `src/styles/theme.css`, `src/styles/globals.css`, and `src/index.css`.


- Store page updated with a stronger ember / flare boutique theme and Etsy-ready presentation.
